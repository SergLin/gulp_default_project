************************************
Инструкция по скачиванию и установке
************************************

Копируем в директорию проекта файлы и папки;
В консоле вводим команду которая скачает и установит "gulp" и все нужные модули для "npm";

$ npm i 

В консоли вводим команду которая скачает и установит все нужные модули для "bower";

$ bower i

В консоли вводим команду которая запускает сборщик "gulp";

$ gulp
